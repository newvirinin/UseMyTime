# UseMyTime - Система управления временем и проектами

## Описание проекта

UseMyTime — это веб-приложение для управления временем и проектами, разработанное для отдела информационно-технического обеспечения ООО НПП «ТЭК». Система позволяет сотрудникам отслеживать время работы над проектами, а руководителям — контролировать и анализировать производительность команды.

## Основные функции

### Для сотрудников
- Создание и управление проектами
- Отслеживание времени с помощью таймера
- Управление задачами в проектах
- Отправка проектов на проверку

### Для руководителей
- Просмотр и проверка проектов подчиненных
- Генерация отчетов по сотрудникам и отделам
- Анализ производительности команды

### Для директоров
- Просмотр отчетов по всей компании
- Статистический анализ и аналитика

## Технологический стек

- **Backend**: Django 5.2.1 (Python)
- **Frontend**: Bootstrap 5, jQuery
- **Database**: PostgreSQL (production), SQLite (development)
- **Authentication**: Django built-in authentication system
- **Reports**: ReportLab for PDF generation

## Установка и настройка

### Требования
- Python 3.8+
- Django 5.2.1
- PostgreSQL 12+ (опционально)

### Шаги установки

1. Клонирование репозитория:
```bash
git clone https://gitlab.com/your-username/usemytime.git
cd usemytime
```

2. Создание виртуального окружения:
```bash
python -m venv venv
source venv/bin/activate  # для Windows: venv\Scripts\activate
```

3. Установка зависимостей:
```bash
pip install -r requirements.txt
```

4. Настройка переменных окружения:
```bash
cp UseMyTime/.env.example UseMyTime/.env
# Отредактируйте .env файл с вашими настройками
```

5. Применение миграций:
```bash
python UseMyTime/manage.py migrate
```

6. Создание суперпользователя:
```bash
python UseMyTime/manage.py createsuperuser
```

7. Запуск сервера:
```bash
python UseMyTime/manage.py runserver
```

## Роли пользователей

1. **Гость** - может просматривать главную страницу и входить в систему
2. **Сотрудник** - управляет своими проектами и временем
3. **Начальник отдела** - проверяет проекты подчиненных
4. **Генеральный директор** - просматривает отчеты по компании
